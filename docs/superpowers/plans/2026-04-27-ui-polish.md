def set_density_mode(self, mode: str):

- [ ] **Step 2: Make timer text fit without changing panel height**

Update `_fit_time_font()` so it only changes the label font and uses a stable label height:

```python
def _fit_time_font(self):
    width = self.time_label.width()
    if width <= 0:
        return
    text = self.time_label.text() or "--:--:--"
    base_size = 28 if self._density_mode == "comfortable" else 24
    for point_size in range(base_size, 13, -1):
        font = QFont("Consolas", point_size)
        font.setBold(True)
        if QFontMetrics(font).horizontalAdvance(text) <= max(80, width - 6):
            self.time_label.setFont(font)
            self.time_label.setFixedHeight(40 if self._density_mode == "comfortable" else 34)
            return
```
